Migrating From Loft Deploy

  1. ./bin/migrate.php <PATH TO LOFT DEPLOY CONFIG>
  2. Move the config.yml file to the appropriate location.
  3. Fill in @todo appearing in the new config.
  4. Rewrite any hooks as processors.
